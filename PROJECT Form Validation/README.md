# Form Validation Project

## Overview

This project is a simple, responsive registration form built with HTML, CSS (using Tailwind CSS), and JavaScript. It provides a clean and user-friendly interface for user account creation, with a focus on accessibility and modern web design principles.

## Features

- **Responsive Design**: The form adapts to different screen sizes using Tailwind CSS utility classes.
- **Form Validation**: (Currently in development) Client-side validation for name, email, and password fields.
- **Modern UI**: Clean, professional design with focus states and smooth transitions.
- **Accessibility**: Proper labeling, semantic HTML, and keyboard navigation support.

## Technologies Used

- **HTML5**: Semantic markup for the form structure.
- **Tailwind CSS**: Utility-first CSS framework for styling (loaded via CDN).
- **JavaScript**: (Planned) For form validation and interactivity.
- **Meta Tags**: SEO and viewport optimization.

## Project Structure

```
/
```

## How to Run

1. Clone the repository:
   ```
   git clone https://github.com/Ojas-1008/Validation-Form.git
   ```

2. Navigate to the project directory:
   ```
   cd "PROJECT Form Validation"
   ```

3. Open `index.html` in your web browser.

Since this is a static HTML file, no build process or server is required. Simply open the file in any modern web browser.

## Form Fields

- **Name**: Text input for the user's full name (required)
- **Email**: Email input for the user's email address (required)
- **Password**: Password input for account security (required)

## Styling

The form uses Tailwind CSS classes for:
- Layout: Flexbox for centering
- Colors: Gray background, white form, indigo accents
- Typography: Bold headings, appropriate font sizes
- Interactions: Focus rings, hover effects, button animations
